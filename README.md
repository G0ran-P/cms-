# cms++ v1
Jednostavan CRUD s ulogama i dozvolama za rad s više korisnika, baziran na laravel 8 frameworku.\
Zahtjeva PHP 7.4, Spatie, LaravelCollective/html, laravel/ui.\
Instalacija preko composera: 

php artisan migrate\
php artisan db:seed \
php artisan serve\
